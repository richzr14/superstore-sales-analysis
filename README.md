# Superstore Sales Analysis (2014–2017)

## Project Overview
Analysed 9,994 US retail orders to identify profit drivers, 
loss-making products and seasonal trends using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualisation

## Dataset
- Source: Kaggle — Sample Superstore Dataset
- 9,994 rows | 21 columns | 2014–2017

## Key Findings
- West region leads in profit ($47K) — South region underperforms
- Tables sub-category loses $11K in profit due to over-discounting
- Technology category delivers the strongest profit margins
- Sales peak in Sep, Nov, Dec — strong Q4 seasonality

## Business Recommendations
1. Cap Furniture discounts at 20% to eliminate loss-making orders
2. Increase Technology inventory and marketing in South region
3. Launch Q1 promotions to smooth revenue seasonality
